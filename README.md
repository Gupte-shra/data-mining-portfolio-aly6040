# Data Mining Portfolio — ALY 6040

A portfolio of data mining projects spanning **classification, clustering, regression, and natural language processing**, completed for **ALY 6040: Data Mining Applications** at Northeastern University (Fall 2024). Each module pairs a real dataset with end-to-end analysis — from EDA through modeling and interpretation.

## Modules

| # | Topic | Technique | Dataset | Highlights |
|---|-------|-----------|---------|------------|
| **1** | [Dataset Selection & EDA](01-dataset-selection-eda) | Exploratory Data Analysis | Employee Performance (100K rows) | Business framing, data validation, EDA |
| **2** | [Classification](02-classification-decision-trees) | Decision Trees | Mushrooms (edible vs. poisonous) | Categorical encoding, tree-based classification |
| **3** | [Clustering](03-clustering) | K-Means / Unsupervised | Obesity levels (eating habits & physical condition) | Segmentation, cluster profiling |
| **4** | [Regression Practice](04-regression-technique-practice) | Predictive Modeling | Student Performance Factors | Feature impact on academic outcomes |
| **5** | [NLP](05-nlp) | Text Mining | Text corpora | Tokenization, text preprocessing, analysis |
| **6** | [Final Report](06-final-report) | Synthesis | Employee Performance | Capstone findings & HR recommendations |

## Skills Demonstrated

- **Supervised learning** — decision tree classification, predictive modeling, evaluation (classification report, ROC-AUC)
- **Unsupervised learning** — clustering and segment profiling
- **NLP** — text preprocessing, tokenization, and analysis
- **Data preparation** — encoding, scaling, pipelines (`ColumnTransformer`, `Pipeline`)
- **Communication** — translating analysis into business-ready reports

## Tech Stack

`Python` · `pandas` · `NumPy` · `scikit-learn` · `NLTK` · `Matplotlib` / `Seaborn` · `Jupyter`

## Repository Layout

Each module folder contains the relevant notebook(s) (`.ipynb`), dataset (`.csv`), rendered analysis (`analysis.md` + charts), and the graded report (`.pdf`).
